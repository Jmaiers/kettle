# kettle

**Get Started**
```
yarn install
```

**Run Dev Environment**
```
npm run dev
```

**Create Production Build**
```
npm run Build
```

**Run Production Build**
```
npm run start
```


### Tests

We want to strive to make every party of this application testable.


**Run Mocha Tests**
```
npm run test
```

**Run Linter**
```
npm run lint
```

**Run Sass Linter**
```
npm run sass-lint
```

**Run All Tests**
```
npm run tests
```


`npm run lint` and `npm test`
